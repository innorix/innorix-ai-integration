# Configuration

The INNORIX AI Integration uses three configuration files.

## Agent Configuration

```text id="w5sxf9"
agent/agent.yaml
```

Defines the runtime configuration for the INNORIX Agent.

Configuration includes:

* Agent information
* Platform connection
* Storage
* Transfer
* Logging
* Health check
* Security

---

## Transfer Configuration

```text id="xglp6b"
config/transfer.example.yaml
```

Defines the data transfer workflow.

Configuration includes:

* Transfer mode
* Source
* Destination
* File selection
* Transfer options
* Performance
* Completion
* Logging

---

## Pipeline Configuration

```text id="zvf0zv"
config/pipeline.example.yaml
```

Defines the AI pipeline integration.

Configuration includes:

* Pipeline provider
* API endpoints
* Authentication
* Trigger options
* Monitoring
* Output
* Logging
